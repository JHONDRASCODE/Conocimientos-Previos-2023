ETIQUETAS HTML5

    Estructura básica:


  • !DOCTYPE html: Define el tipo de documento como HTML5.

  • html: Define el inicio del documento HTML.

  • head: Contiene información sobre el documento, como el título y los metadatos.

  • title: Define el título del documento.

  • meta: Define metadatos del documento, como la codificación de caracteres y la descripción.

  • body: Contiene el contenido principal del documento.

  • /html: Define el final del documento HTML.

    • Elementos de contenido:


  • h1 a h6: Definen títulos de diferentes niveles.

  • p: Define un párrafo de texto.

  • br: Inserta un salto de línea.

  • img: Inserta una imagen.

  • a: Define un enlace a otra página web o recurso.

  • ul: Define una lista sin orden.

  • ol: Define una lista ordenada.

  • li: Define un elemento dentro de una lista.

    Elementos multimedia:


  • video: Inserta un video.

  • audio: Inserta un audio.

  • iframe: Inserta un contenido embebido, como un video de YouTube.

  • Elementos de formulario:

  • form: Define un formulario.

  • input: Define un campo de entrada de datos, como un campo de texto o un botón.

  • label: Define una etiqueta para un campo de entrada.

  • select: Define una lista desplegable.

  • option: Define una opción dentro de una lista desplegable.

    Elementos semánticos:


  • header: Define el encabezado de la página.

  • nav: Define la navegación de la página.

  • section: Define una sección de contenido.

  • article: Define un artículo independiente.

  • aside: Define contenido secundario.

  • footer: Define el pie de página.

  • Otras etiquetas importantes:

  • meta charset="utf-8": Define la codificación de caracteres del documento.

  • link: Define una referencia a un archivo externo, como una hoja de estilo CSS.

  • script: Define un script JavaScript.
