HOJAS DE ESTILO EN CASCADA CSS

En esta introducción, te familiarizarás con los conceptos básicos de CSS3 y algunos de sus temas más importantes:

Tipos de selectores

  • Selectores por tipo (p. ej., h1, div)
  • Selectores por clase (p. ej., .clase)
  • Selectores por ID (p. ej., #id)
  • Selectores descendientes (p. ej., div p)
  • Selectores de hermanos adyacentes (p. ej., h1 + h2)

    • Pseudo clases

  • Pseudo clases de estado (p. ej., :hover, :active)
  • Pseudo clases de posición (p. ej., :first-child, :last-child)

    • Unidades de medida relativas

  • Porcentajes ( %): se basan en el tamaño del elemento padre.
  • Em: se basan en el tamaño de la fuente del elemento padre.
  • Rem: se basan en el tamaño de la fuente del elemento raíz.

    • Fondos y bordes

  • Fondos: puedes agregar imágenes, colores y degradados como fondo de un elemento.
  • Bordes: puedes controlar el grosor, estilo y color de los bordes de un elemento.

    • Modelo de cajas

  • Caja de contenido: contiene el contenido del elemento.
  • Relleno: área entre el contenido y el borde.
  • Borde: área entre el relleno y el margen.
  • Margen: área entre el borde y el elemento adyacente.

    • Posicionamiento

  • Posicionamiento estático: el elemento se coloca en su posición normal en el flujo del documento.
  • Posicionamiento relativo: el elemento se desplaza desde su posición normal en el flujo del documento.
  • Posicionamiento absoluto: el elemento se coloca en una posición específica en relación con su elemento padre posicionado.
  • Posicionamiento fijo: el elemento se coloca en una posición fija en la pantalla.

    • Display

  • Block: el elemento se muestra como un bloque rectangular.
  • Inline: el elemento se muestra en línea con otros elementos.
  • None: el elemento no se muestra.

    • Responsive Design “Media Queries”

  • Permite que tu página web se adapte a diferentes dispositivos y tamaños de pantalla.
  • Puedes usar media queries para cambiar el diseño de tu página web en función de la anchura de la pantalla, la altura de la pantalla, la orientación del dispositivo y la resolución de la pantalla.
