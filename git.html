Github


Comandos utilizados de GIT.


    Configuracion:

  • git init: Inicializa un repositorio Git en el directorio actual
  • git clone: Clona un repositorio remoto (como GitHub) a tu computadora.
  • git config: Configura tu nombre de usuario y correo electrónico para Git.

    • Gestión de cambios:

  • git add: Agrega archivos al área de preparación para el siguiente commit.
  • git commit: Guarda los cambios en el área de preparación como un commit.
  • git push: Envía tus commits locales al repositorio remoto.
  • git pull: Obtiene los últimos cambios del repositorio remoto y los fusiona con tu copia local.

    • Navegación:

  • git status: Muestra el estado actual de tu repositorio, incluyendo los archivos modificados y sin confirmar.
  • git log: Muestra un historial de los commits realizados en el repositorio.
  • git checkout: Cambia entre diferentes ramas de tu repositorio.
  • git branch: Crea una nueva rama o cambia a una rama existente.

    Deshacer cambios:

  • git reset: Deshace los cambios sin eliminarlos del historial.
  • git revert: Deshace un commit y elimina sus cambios del historial.

    Colaboración:

  • git merge: Combina los cambios de dos o más ramas.
  • git pull request: Envía una solicitud de pull para que tus cambios sean revisados y fusionados en otra rama.
